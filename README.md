## Start develop

1. install `pnpm`

```bash
npm install -g pnpm
```

2. install node_modules

```bash
pnpm install
```

3. local server

```bash
pnpm dev
```

4. build

```bash
pnpm build
```

Open [http://localhost:5731](http://localhost:5731) with your browser to see the result.
